What is Python?

Ang simple, madaling matutunang syntax ng Python ay nagbibigay-diin sa pagiging madaling mabasa at binabawasan ang cost ng pagpapanatili ng program.

Ang mataas na antas na built in na mga structure ng data na sinamahan ng dynamic na pag-type at dynamic na pag-binding, ay ginagawa itong talagang kaakit-akit para sa Rapid Application Development, pati na rin para sa paggamit bilang isang scripting o glue language para ikonekta ang mga kasalukuyang bahagi nang magkasama.
