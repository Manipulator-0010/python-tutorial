About Python Programming

Kadalasan, ang mga programmer ay nagugustuhan ang Python dahil sa pagtaas ng produktibidad na ibinibigay nito. Dahil walang step sa pagsasama-sama, ang ikot ng pag-edit-test-debug ay napakabilis.

Sinusuportahan ng Python ang mga module at package, na naghihikayat sa modularity ng program at muling paggamit ng code. Ang Python interpreter at ang malawak na standard library ay available sa source o binary form nang walang bayad para sa lahat ng pangunahing platform, at maaaring malayang ipamahagi.
